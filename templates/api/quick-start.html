Quick Start

Last updated: April 12th, 2022

Introduction

This is a project work presented by Okwor Dennis Ebuka to the department of Computer and Information Science, Faculty of Applied Natural Science, Enugu State University of Science and Technology. The project involves the Design and Implementation Of Open API (a case study of COVID-19). The data used in developing this project was obtained from the public data provided by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University.

Quick Start

API Endpoint

The API Endpoint provides an enpoint url which can be queried for COVID-19 data.

Base API Endpoint:

{{request.get_host}}/api/covid-data/

Query Parameters

The API endpoint can be queried with the following parameters. The type of data returned and the query parameters are listed in the table below.

Queries and Return Data
# Query Parameter Data Returned
1 fips
2 admin2 Thornton
3 province_state
4 country_region
5 data_from
6 data_to
7 lenght

Returned Data Format

Data returned is in JSON (JavaScript Object Notation) by default.

Code Sample

The code sample below are given in different programming language but can also be used following the API consumer perference.

Useful Tip:

You can use this online Language entity encoder/decoder to generate your code examples.

HTML Code Example

 '{{request.get_host}}/api/covid-data/',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_ENCODING => '',
  CURLOPT_MAXREDIRS => 10,
  CURLOPT_TIMEOUT => 0,
  CURLOPT_FOLLOWLOCATION => true,
  CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
  CURLOPT_CUSTOMREQUEST => 'POST',
  CURLOPT_POSTFIELDS => 'lenght=10&country_region=Ghana',
  CURLOPT_HTTPHEADER => array(
    'Content-Type: application/x-www-form-urlencoded'
  ),
));

$response = curl_exec($curl);

curl_close($curl);
echo $response;

PHP cURL Code Example

 '{{request.get_host}}/api/covid-data/',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_ENCODING => '',
  CURLOPT_MAXREDIRS => 10,
  CURLOPT_TIMEOUT => 0,
  CURLOPT_FOLLOWLOCATION => true,
  CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
  CURLOPT_CUSTOMREQUEST => 'POST',
  CURLOPT_POSTFIELDS => 'country_region=Nigeria&data_from=2020-06-21&data_to=2021-06-21',
  CURLOPT_HTTPHEADER => array(
    'Content-Type: application/x-www-form-urlencoded'
  ),
));

$response = curl_exec($curl);

curl_close($curl);
echo $response;
